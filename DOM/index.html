DOM => Document Object Model

The DOM is a programming interface that allows JavaScript to interact with and manipulate HTML and CSS of a web page. It represents the structure of a web page as a tree-like hierarchy of objects.

How JavaScript Accesses HTML:

  1. Window Object
    • The window is the global object in the browser environment.
    • It represents the browser window/tab.
    • It is not part of JavaScript, but provided by the browser.
  2. Document Object
    • window contains many properties and methods, one of which is document.
    • document represents the entire HTML page loaded in the browser.
    • Through document, you can access and manipulate:
      • Elements (getElementById, querySelector)
      • Content (innerHTML, textContent)
      • Styles (style)
      • Events (addEventListener)
